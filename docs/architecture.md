# Food Independence Simulator Architecture

## Purpose

Food Independence Simulator is a long-term scientific research project focused on evaluating whether Japan can maintain food supply using domestic resources if food imports cease.

## Future architecture direction

The project will evolve into a modular simulation platform with clearly separated responsibilities for data, domain logic, simulation orchestration, and presentation.

## Why the system is modular

A modular structure makes the project easier to evolve over time, supports independent testing, and keeps future scientific models isolated from the user interface and application shell.

## Future expansion areas

- Region
- Land
- Climate
- Soil
- Crop
- Livestock
- Resource circulation
- Food balance
